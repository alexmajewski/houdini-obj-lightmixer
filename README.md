# Houdini /obj/ Light Mixer

Obj Light Mixer is a custom panel built to mimic the Solaris Light Mixer functionality and appearance (currently only the "Sliders" tab is reconstructed -- this may change in the future).

### Supported engines

Currently only Redshift is supported.

## Installation

### Option 1: Install as package

### Option 2: Copy .pypanel file into User Preferences Folder
